# 研究开发计划

**文档版本**: v1.0.0
**状态**: Release
**更新日期**: 2026-04-10

## 1. 项目概述
本项目旨在开发一套“管道机器人控制系统（Pipeline Robot Control System）”，包含基于 Electron 的上位机控制端和基于 Python 的下位机三维物理模拟器。

### 2.1 关键里程碑
*   **里程碑 ① (04-10)**：完成需求与架构基线化，双独立链路（TCP+UDP）方案定型。
*   **里程碑 ② (04-30)**：完成核心代码编写，包括自定义 CRC32 协议解析、OpenGL 3D 渲染、OpenCV 视频流管线。
*   **里程碑 ③ (06-30)**：系统整体打包发布（Electron Dir 绿色包 + PyInstaller EXE），交付最终用户。
